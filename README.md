# PGE Analyse

A React application for analyzing PGE reports with two main features:
- **Report Analyse**: Upload and analyze PGE reports
- **Compare Plan**: Compare different plans (coming soon)

## Getting Started

### Installation

Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Build for production:
```bash
npm run build
```

## Features

### Report Analyse Tab
- Upload multiple PGE reports (starts with 3 slots)
- Automatically adds a 4th slot when the 3rd report is uploaded
- File preview with name and size
- Remove uploaded files
- Submit button to send reports to backend (ready for backend integration)
